# A Hidden Markov Model-Based Approach for the Automatic Classification and Annotation of Kunitz BPTI Domains

This repository provides the complete pipeline and materials used to build, evaluate, and validate a 

profile HMM for detecting Kunitz-type domains in protein sequences. The project was developed as the 

final assessment for the Laboratory of Bioinformatics 1 course at the University of Bologna.

# Project Overview

Profile Hidden Markov Model for the classification of Kunitz-type protease inhibitor domains. Includes 

structural data curation from PDB, MMseqs2 clustering, PDBeFold alignment, HMM construction with 

HMMER, and rigorous benchmarking on Swiss-Prot datasets. Final project for the Laboratory of 

Bioinformatics course at the University of Bologna.
